## ▌ O que é

Coletânea de **todos os códigos em Java** que escrevi durante os estudos — dos primeiros `Hello World` a exercícios mais elaborados como **ALU** e **calculadora geral**. Cada pasta é um conjunto de atividades (ATV).

## ▌ Estrutura

```
JAVA-todos-os-codigos-em-java/
├── T-0==1/      # atividades iniciais (T01-1, T01-2…)
├── T0-1/        # ATV 1 a 5
├── T0-2/        # ATV 1 a 5  →  inclui ALU.java e Calculadora_geral.java
├── T0-3/        # ATV 1 a 7
└── T0-4/        # ATV 1 a 5
```

Cada `ATV - N/` contém um `main.java` com a solução daquela atividade.

### Destaques
- `T0-2/ATV - 4/ALU.java` — implementação de uma **Unidade Lógica e Aritmética** em Java.
- `T0-2/ATV - 5/Calculadora_geral.java` — calculadora com operações gerais.

## ▌ Como compilar e rodar

```bash
# dentro da pasta da atividade desejada
javac main.java
java Main      # ou o nome da classe pública
```

> Requer o **JDK** instalado (`javac` e `java` no PATH).
